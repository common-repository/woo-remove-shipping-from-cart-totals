Simple WooCommerce Plugin which makes the shipping cost not be calculated in cart totals.

== Description ==
This plugin helps you remove the Shipping cost from the cart totals.

It just removes the shipping from cart totals, however, it is still displayed to the customer.

Often times, such a set up is needed for some stores so this plugin comes in handy when needed.

The plugin works just by being activated.

Tested & works with Gutenberg + the latest version of WooCommerce.

= Docs & Support =
You will find more information about the plugin at the plugin [website](https://justentrepreneurship.com)

== Installation ==

= From your WordPress dashboard =

Go to Dashboard > Plugins > Add New
Search for 'WooCommerce Remove Shipping from Cart Totals'
Click install and then Activate the plugin

= From WordPress.org =

Download the package from wordpress.org Plugin Directory
Unzip the files
Upload the unzipped files with FTP to the plugin directory in your wordpress installation
Go to Dashboard > Plugins to activate

== Frequently Asked Questions ==

What themes have this plugin been tested on?

This plugin has been tested and created on a test environment of the Divi theme with Gutenberg enabled & the latest version of WooCommerce.
